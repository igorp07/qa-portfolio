# QA Portfolio

This repository contains a Quality Assurance portfolio with practical examples of bug reports, test cases, and test artifacts created for learning and professional demonstration purposes.

## Repository Structure

- **bug-reports/**
  - `BUG-001-checkout-allows-purchase-without-name.md`  
    Checkout allows proceeding without required customer information.

- **test-cases/**
  - `CT-001-checkout-required-fields-validation.md`  
    Validates required fields on checkout and detects BUG-001.

## Notes

- All artifacts in this repository were created for QA practice and portfolio purposes.
- The scenarios are based on realistic system behavior and common validation rules.
- Bug reports and test cases are intentionally linked to demonstrate traceability.
